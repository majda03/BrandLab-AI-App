# BrandLab-AI-App
AI-inspired branding tool that generates visual identities, color systems, typography pairings, brand strategy concepts and export-ready CSS variables.
BrandLab AI is a frontend application designed to generate complete visual brand directions.

Users can create branding concepts by selecting industry, mood and target emotion. The application generates color systems, typography recommendations, brand personality keywords, strategic positioning notes, interactive UI previews and export-ready CSS variables.

The project explores the intersection of frontend development, branding and visual storytelling.
